# Nexus Notifications Microservice

Notifications microservice for NexusERP system.

## Technologies
- Java 21
- Spring Boot 3.3.0
- Spring Data JPA
- PostgreSQL
- RabbitMQ (AMQP)
- SpringDoc OpenAPI

## How to run
1. Navigate to the `NOTIFICATIONS` directory.
2. Run `mvn clean package`.
3. Run `docker-compose up`.
