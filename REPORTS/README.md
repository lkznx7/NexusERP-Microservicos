# REPORTS Microservice

Reports microservice for NexusERP.

## Technologies
- Java 21
- Spring Boot 3.3.0
- PostgreSQL
- RabbitMQ
- Actuator
- SpringDoc OpenAPI

## How to run
1. Build the project: `mvn clean package`
2. Run with Docker: `docker-compose up -d`
3. Access Swagger UI: `http://localhost:8091/swagger-ui.html`
