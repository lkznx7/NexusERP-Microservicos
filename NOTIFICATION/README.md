# NOTIFICATION - NexusERP

### Sobre

Projeto desenvolvido exclusivamente para estudo de arquitetura de software moderna.

### Tecnologias

* Java 21
* Spring Boot
* PostgreSQL
* RabbitMQ
* Docker

### Observação

Este projeto utiliza Inteligência Artificial apenas para acelerar tarefas repetitivas e geração de código boilerplate.

Toda modelagem de domínio, arquitetura, regras de negócio e implementação das funcionalidades são realizadas manualmente para fins de aprendizado.
