# Sales Microservice

This is the Sales microservice for the Nexus ERP system.

## Technologies
- Java 21
- Spring Boot 3.2.2
- PostgreSQL
- Maven
- Docker

## Running with Docker
```bash
docker-compose up --build
```

## API Documentation
The API documentation will be available at `http://localhost:8083/swagger-ui.html` once the service is running.
